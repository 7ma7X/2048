## How to play

```
cargo build
./target/debug/rust2048
```

Or simply

|  KEY  |  manipulation  |
| ---- | ---- |
|  i / I |  move up  |
|  m / M |  move down  |
|  j / J |  move left  |
|  k / K |  move right  |

## How to test

```
cargo test
```
